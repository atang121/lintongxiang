# 2026-05-01 工作记录

## 安全与上线前修复

- 修复短信 preview 生产风险：`NODE_ENV=production` 且短信未完整配置时不返回验证码。
- 加固上传：图片 MIME、扩展名、category 白名单和路径安全。
- 生产空库不自动灌入 demo 数据。
- 预约查询、创建、完成、取消、失败按当前登录身份校验。
- 公开用户详情脱敏。
- 管理物品路由顺序修复。
- 恢复用户状态时同步恢复其被系统下架的物品。

## 用户体验

- 新用户欢迎弹窗改为“第一分钟”轻引导。
- 发布需求可无图，无图用 `WantedCover`。
- 详情页图片改为 `ItemImageCarousel`，支持多图滑动和横竖图统一展示。
- 详情页移动端隐藏全局底部导航，显示详情专属动作条。
- 收藏、分享增加明确反馈；微信内分享给出右上角菜单提示，非微信支持复制链接或系统分享。
- 新增 PWA manifest、`/app-icon.svg` 和首页“添加到桌面”提示。

## 预约、私信、通知

- 邻里等级改为交换次数等级：`新邻居`、`有过交换`、`靠谱邻居`、`热心邻居`。
- 交换完成只能由发布者确认；预约者可取消预约、联系、提醒确认。
- 消息页可打开私信会话并回复，打开会话时标记对方来信已读。
- 后端发送私信不再创建 `type='message'` notification。
- 前端通知页过滤历史 message 通知，通知只承接预约/系统类内容。

## 飞书与运维

- 反馈投递支持飞书机器人、飞书多维表格、邮件兜底，并返回投递诊断字段。
- 新增本机运维面板 `tools/local-ops-dashboard/`。
- 新增 SQLite 备份脚本和 `server` 的 `backup:sqlite` 命令。
