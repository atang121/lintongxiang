# 邻里童享上线版说明

本目录用于存放“适合正式交付、部署、归档”的项目文件副本。

## 目录说明

- `方案书.md`
  给老板审核的完整项目方案书
- `_SETUP_GUIDE.md`
  当前接入与部署说明
- `docs/`
  当前 Demo 素材来源等补充资料
- `h5/`
  前端源码与配置模板
- `server/`
  后端源码、编译产物、配置模板与当前数据库快照

## 已排除内容

为了让交付包更干净，本目录默认不包含以下内容：

- `.git`
- `node_modules`
- `.next`
- 本机缓存、隐藏系统文件
- 真实密钥环境文件

## 关于环境变量

本目录保留的是配置模板：

- `h5/.env.local.example`
- `server/.env.example`

正式上线时，请将现网密钥与参数填入对应环境变量，不建议把真实密钥直接复制进交付包长期保存。

## 关于数据库

`server/data/swap.db` 为当前项目的数据库快照，适合本地复现与演示。

如果后续进入公网正式运营，建议：

1. 将核心业务数据迁移到云数据库
2. 飞书多维表格继续作为运营配置与审核后台
3. 数据库快照仅保留为备份或演示环境使用
