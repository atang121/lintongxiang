本目录用于存放“适合正式交付、部署、归档”的项目文件副本。

当前代码对应 2026-05-05 试运营前整理版本：在 2026-05-01 安全加固基础上，继续完成用户服务协议、协议版本确认、候补预约、私信内预约、反馈处理回复、敏感词/活物禁发、管理员删除原因通知、协议管理工具和后台通知运营优化。

## 目录说明

- `方案书.md`
  给老板审核的完整项目方案书
- `_SETUP_GUIDE.md`
  当前接入与部署说明

  架构、集成、运维、阶段交接与 Demo 素材来源

- `server/`
  Express API 源码、配置模板与 SQLite 数据库

## 已排除内容

为了让交付包更干净，本目录默认不包含以下内容：

- `.git`
- `node_modules`
- `.next`
- 本机缓存、隐藏系统文件
- 真实密钥环境文件

## 快速验证

```bash
cd h5
node --test test/*.mjs
npm exec tsc -- --noEmit
npm run build
```

```bash
cd server
npm run build
npm run backup:sqlite
```

本机可视化运维面板：

```bash
node tools/local-ops-dashboard/server.mjs
```

打开 `http://127.0.0.1:4318`，可查看 PM2、启动/重启前后端、执行 SQLite 备份和健康检查。

## 关于环境变量

本目录保留的是配置模板：

- `h5/.env.local.example`
- `server/.env.example`

正式上线时，请将现网密钥与参数填入对应环境变量，不建议把真实密钥直接复制进交付包长期保存。

## 关于数据库

`server/data/swap.db` 为当前项目的数据库快照，适合本地复现与演示。

当前核心数据仍是 SQLite。5 个小区冷启动可以继续使用，但正式运营时至少要保证每日备份；上线当天建议先执行一次手动备份。备份脚本为：

```bash
cd server
npm run backup:sqlite
```

如果后续进入长期公网运营，建议：

1. 将核心业务数据迁移到云数据库
2. 飞书多维表格继续作为运营配置与审核后台
3. 数据库快照仅保留为备份或演示环境使用
